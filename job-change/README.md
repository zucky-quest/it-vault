# Job Change Notes

転職活動で利用するMarkdownベースの情報管理用ディレクトリです。

## 役割

- 職務経歴書の元データ管理
- 面接準備メモの保存
- 応募企業ごとの情報整理

## 推奨ファイル構成

- `profile.md` - 基本プロフィール・強み・転職希望軸
- `career-history.md` - 職務経歴・実績・技術スタック
- `resume-draft.md` - 職務経歴書向けの本文
- `self-pr.md` - 転職サイト用の自己PR文
- `interview-notes.md` - 面接メモ・質問対策
- `applications/` - 応募企業ごとのメモ
